# Алгоритм Дейкстры (курсовой проект)

GUI-приложение на Python для поиска кратчайших путей в графе алгоритмом Дейкстры.

## Ввод
- n — число вершин (0..n-1)
- directed — ориентированный/неориентированный граф
- s — стартовая вершина
- t — целевая вершина
- рёбра в формате строк: `u v w`

## Реализованные методы
- Алгоритм Дейкстры
- Приоритетная очередь
- Пошаговый лог: извлечение из очереди, посещение, релаксации

## Вывод
- визуализация графа и весов на рёбрах
- экспорт лога в .txt

## Запуск
```bash
pip install -r requirements.txt
python run.py
